# FinWise – Smart Financial Decision Support System

FinWise is a web application designed to help users analyze financial habits, track goals, and make smarter purchase decisions using real-time data and intelligent fallback mechanisms.

## Features
- User Authentication (Login & Signup using Firebase)
- Financial Assessment with scoring system
- Goal Tracker with progress tracking
- Purchase Advisor (API + manual fallback)
- Smart UI with loading effect

## Technologies Used
- HTML, CSS, JavaScript
- Firebase Authentication
- Firestore Database
- RapidAPI

## How to Run
1. Open using Live Server
2. Add Firebase config in firebase.js
3. (Optional) Add API key in advisor.js
4. Run index.html
